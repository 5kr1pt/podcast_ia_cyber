# Prompt para Arte da Capa (Foco: Cyber & Corporativo)

```
Crie uma imagem quadrada 1:1, >> minimalista <<, para a capa de um podcast sobre cibersegurança no setor financeiro.

Descrição da Imagem: Uma visualização de dados digitais, com linhas de luz formando uma estrutura complexa que se assemelha a um escudo ou a um cofre digital. O design deve ser limpo, elegante e futurista. Adicione elementos comuns em um podcast, como um microfone.

Estilo e Adjetivos: Arte conceitual, corporativo, futurista, minimalista, iluminação cinematográfica, visualização de dados, abstrato, elegante, alta tecnologia (high-tech).

Paleta de Cores: Utilize predominantemente tons de azul escuro e branco. Adicione detalhes sutis e linhas de luz na cor vermelha, para criar um contraste sofisticado.
```
